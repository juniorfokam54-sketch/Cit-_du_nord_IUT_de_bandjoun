## 🖼️ Configuration de l'Image de Prévisualisation

### Option 1 : Utiliser une image existante
1. Choisissez une belle photo représentative (ex: `images/FRATERNITE/photo3.jpg`)
2. Renommez-la en `preview.jpg`
3. Placez-la dans le dossier `images/`
4. Mettez à jour les URLs dans les balises meta si nécessaire

### Option 2 : Créer une image personnalisée
1. Ouvrez `preview-generator.html` dans votre navigateur
2. Prenez une capture d'écran (1200x630 pixels recommandé)
3. Enregistrez l'image comme `preview.jpg` dans le dossier `images/`
4. Utilisez un outil en ligne comme Canva pour créer une image professionnelle

### Option 3 : Générer automatiquement
Utilisez des outils en ligne comme :
- [Open Graph Image Generator](https://opengraph.xyz/)
- [Social Image Generator](https://social-image-generator.vercel.app/)

## 🌐 Hébergement

### Préparation avant l'hébergement :
1. **Remplacez les URLs** dans les balises meta par votre domaine réel
2. **Ajoutez l'image de prévisualisation** dans le dossier `images/`
3. **Testez les liens** pour vous assurer qu'ils fonctionnent

### Plateformes recommandées :
- **GitHub Pages** (gratuit)
- **Netlify** (gratuit avec domaine personnalisé)
- **Vercel** (gratuit)
- **Firebase Hosting** (gratuit)

### Étapes pour GitHub Pages :
1. Créez un repository GitHub
2. Uploadez tous les fichiers
3. Allez dans Settings > Pages
4. Sélectionnez "main branch" comme source
5. Votre site sera disponible à `https://votre-nom.github.io/nom-du-repo/`
